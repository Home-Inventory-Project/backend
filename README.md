# Home Inventory backend

<!-- ABOUT THE PROJECT -->
## About The Project

This is the backend service to manage a HomeInventory API.

The main purpose of the project is to keep track of the products
that you buy when you go shopping for food or other home product (with expiry date and other details).
In  that way you can keep track more consciously what you have at home and avoid waste.


### Built With

* [ Apollo Graphql ](https://www.apollographql.com/)
* [ Sequelize ](https://sequelize.org/)
* [ Sqlite3 ](https://www.sqlite.org/index.html)

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Home-Inventory-Project/backend.git
   ```

   ```sh
   npm install
   ```
